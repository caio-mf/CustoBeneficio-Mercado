# CustoBeneficio-Mercado
Em vez de uma lista de compras comum, fiz um script que calcula o custo-benefício real de produtos no supermercado (porcentagem de preço vs peso)

Este é um projeto prático que criei logo no início dos meus estudos em Python para resolver um problema real do dia a dia: descobrir qual embalagem ou produto vale mais a pena levar no supermercado. 

## 🚀 Como funciona
O script recebe o peso (em gramas) e o valor de dois produtos diferentes, faz o cálculo matemático para descobrir o preço proporcional por quilo e exibe os resultados na tela de forma simples e direta.

Desenvolvi este código de forma autônoma para praticar:
* Entrada e saída de dados pelo terminal.
* Operações matemáticas e manipulação de variáveis.
* Resolução de problemas reais com lógica de programação.
